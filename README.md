# youtube_playlist_downlaoder
Python tool to download YouTube playlists as MP3 files with quality options and progress tracking. Perfect for meditation music and audio content.

All files are saved as MP3 with original titles preserved.

## â— Troubleshooting

### Common Issues

1. **"FFmpeg not found"**
   - Solution: Install FFmpeg and add to PATH

2. **"yt-dlp not installed"**
   - Solution: Run `pip install yt-dlp`

3. **Network errors**
   - Solution: Check internet connection and try again
